# Student Event Registration Form

A simple HTML project for student event registration.

## Project Description
This project allows students to register for college events using an online form.  
After successful registration, users are redirected to a success page.

## Features
- Student registration form
- Input fields for student details
- Event selection
- Submit button
- Registration success confirmation page

## Files Included
- `event.html` – Main registration form
- `success.html` – Success page after submission

## Technologies Used
- HTML5

## Purpose
This project was created as a beginner HTML practice project to understand:
- Forms
- Input fields
- Buttons
- Basic webpage structure
- Navigation between pages
